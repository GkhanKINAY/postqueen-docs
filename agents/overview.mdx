## Just talk to her

Message her the way you would message a colleague. She answers inside ChatGPT and the Claude app, reads your WhatsApp and Telegram messages through OpenClaw or Hermes, and she is one command away in your terminal. Tell her what you want the world to see, and she takes care of the rest: the words, the visual and the schedule.

## She runs on your terms

Leave her alone and she is a true autopilot: whatever she puts on the calendar goes out on time without another word from you. Want more say? Everything is visible on the calendar before it publishes, and if you ask for drafts she holds each post until you approve it.

## Install the skill

One command installs her skill into any coding agent that supports the `skills` registry:

```bash
npx skills add GkhanKINAY/postqueen-agent
```

What lands is her `SKILL.md`, a playbook that teaches your agent how she works before it runs a single command. It registers under the name `postqueen` and declares `Bash(postqueen:*)` as the only tool it needs. After reading it, the agent knows:

- **Discovery.** `integrations:list` returns every connected channel with its integration ID, and `integrations:settings` shows what each channel accepts.
- **Dynamic data.** `integrations:trigger` fetches values that cannot be hardcoded, like subreddit flairs, YouTube playlists or LinkedIn company pages.
- **Posting.** `posts:create` takes content, a required ISO 8601 date and integration IDs, with `--settings` for platform specifics and `--json` for full campaigns.
- **Media rules.** Every file goes through `postqueen upload` first, and only the returned URL belongs in `-m`. Media has to live on a PostQueen domain, and the API rejects the request otherwise.

The skill tells your agent to install the CLI if it is missing. To put it in place yourself:

```bash
npm install -g postqueen
```

Next she needs your API key. In the app at [app.postqueen.ai](https://app.postqueen.ai), open **Settings**, then **Developers**, then **Public API**, and click **Reveal**. Keep the key secret: it grants full access to your account.

```bash
export POSTQUEEN_API_KEY=your_api_key
```

<Note>
Self-hosting? Set `POSTQUEEN_API_URL` to your own instance and the CLI talks to it instead of the hosted API.
</Note>

Now your agent can see your channels and schedule its first post:

```bash
# List your connected channels and their integration IDs
postqueen integrations:list

# Schedule a post (-s is required and takes an ISO 8601 date)
postqueen posts:create \
  -c "We just shipped dark mode 🌙" \
  -s "2026-08-01T09:00:00Z" \
  -i "your-integration-id"
```

The full command reference lives in the [command reference](/cli/command-reference).

Every agent session runs the same loop: authenticate, discover channels, prepare media, post, then check the numbers. [Why agents love her](/agents/why-agents) walks all seven steps.

## Connect over MCP

Prefer tool calls over shell commands? PostQueen ships a hosted MCP server with 10 tools, so an MCP client can list your channels, upload media and schedule posts without installing anything.

For Claude Code it is one line:

```bash
claude mcp add --transport http postqueen https://api.postqueen.ai/mcp/YOUR_API_KEY
```

Clients that speak remote HTTP MCP directly, such as Cursor and Codex, take the URL as-is:

```json
{
  "mcpServers": {
    "postqueen": {
      "url": "https://api.postqueen.ai/mcp/YOUR_API_KEY"
    }
  }
}
```

Claude Desktop reaches a remote server through the `mcp-remote` bridge instead. The exact config is on the [Claude Desktop page](/mcp/clients/claude-desktop).

Per-client walkthroughs are in [MCP setup](/mcp/setup), and every tool is described in the [tool reference](/mcp/tools).

## From your chat apps

Message her on WhatsApp or Telegram through OpenClaw or Hermes, and the same route brings her into Slack and Discord. Prefer the browser? [app.postqueen.ai](https://app.postqueen.ai) runs your whole calendar from your phone too. The setup guides live on the [OpenClaw](/agents/openclaw) and [Hermes](/agents/hermes) pages.

## Try saying

Once she is connected, talk to her in plain words:

> *"Turn my latest YouTube video into posts for X, LinkedIn and Threads."*

She turns a request like that into scheduled posts across your channels, and the calendar stays yours to review.
