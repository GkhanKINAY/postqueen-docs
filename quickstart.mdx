## Talk to her

PostQueen answers any agent that can run a skill or call an MCP tool. Grab your API key first: your workspace lives at [app.postqueen.ai](https://app.postqueen.ai), so open **Settings** there, then **Developers**, then **Public API**, and click **Reveal**.

<img src="/images/brand/chat-agent.svg" alt="Just talk to her: one voice note and PostQueen generates the visual and schedules the posts" />

Install the skill and hand your agent the key:

```bash
npx skills add GkhanKINAY/postqueen-agent
export POSTQUEEN_API_KEY="your-api-key"
```

Or connect Claude Code straight to her MCP server:

```bash
claude mcp add --transport http postqueen https://api.postqueen.ai/mcp/YOUR_API_KEY
```

That URL contains your API key, so treat it like a password. The [MCP setup page](/mcp/setup) also shows the header form that keeps the key out of the URL.

That is the whole setup. For the full story, including OpenClaw, Hermes, and self-host endpoints, see the [agents overview](/agents/overview).

## Your first post

<Steps>
  <Step title="Connect a channel">
    In the app, connect at least one channel. PostQueen posts to 30+ networks, so start with the one your audience actually reads.
  </Step>
  <Step title="Ask her for a post">
    From any connected agent, ask in plain language: "Write a launch post for tomorrow at 9am." Or write it yourself in the editor. Either way, the post lands on the schedule.
  </Step>
  <Step title="Review it on the calendar">
    Open the calendar to see what is planned. Edit or delete anything before it goes out. Prefer to sign off on each post? Ask her for drafts and she holds every post until you approve it.
  </Step>
  <Step title="It publishes on time">
    No extra confirmation, no reminder to click. What is on the calendar goes out.
  </Step>
</Steps>
