## PostQueen Cloud: billing and refunds

Write to [support@postqueen.ai](mailto:support@postqueen.ai) from the email address on your account so we can find you quickly, and say what you expected versus what happened. You can close the account yourself from **Settings**, then **Account**, then **Delete account** in the app; email works too, and GDPR requests go through the same inbox. Leave API keys, OAuth secrets and card numbers out of the message, since support does not need them to help you.

Before opening a billing ticket, take a quick look at the [troubleshooting pages](/troubleshooting/billing). Many "I was charged but X" cases turn out to be a Stripe.js script blocker or a private-window localStorage issue.

## 👑 How to write a report that gets a fast answer

A good report gets a good answer. Walk through the four parts below in order.

### Goal

Start with what you were trying to do:

- *I want to schedule a post on Reddit at 6pm*
- *I want to run in a Linux container on a Raspberry Pi*
- *I want to use a custom domain name*

### Environment

Share the relevant parts of your setup. For web app issues, that means your browser. For installation or server issues, that means your OS and how you installed:

- *I'm using Firefox on Windows 10*
- *I'm using a Raspberry Pi 4 with Ubuntu 20.04, and Node version 18*
- *This is a fresh installation on a Mac*

### Changes

If it worked before and broke, something changed. Tell us what:

- *I updated my browser to the latest version and now ...*
- *I upgraded to the latest release and now ...*
- *I think this used to work, but now ...*

### Results

Describe what happened, and what you expected instead:

- *I see a blank screen*
- *I see an error message*
- *I see a 404 page*

That is it. The more of these four you cover, the faster someone can help.
