## Settings Schema

When creating a post for MeWe, use the following settings schema:

```json
{
  "settings": {
    "__type": "mewe",
    "postType": "group",
    "group": "group-id"
  }
}
```

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `__type` | `string` | Yes | Must be `mewe` |
| `postType` | `string` | Yes | Where to post: `timeline` or `group` |
| `group` | `string` | Conditional | Target group ID. Required when `postType` is `group`. |

### `postType`

Where the post is published. Use `timeline` to post to your own MeWe timeline, or `group` to post into a MeWe group (which then requires `group`).

### `group`

The ID of the MeWe group to post into. Required only when `postType` is `group`. Select the group in the PostQueen UI when configuring the MeWe channel to obtain its ID.

## Complete Example

### Timeline Post

```json
{
  "type": "schedule",
  "date": "2024-12-14T10:00:00.000Z",
  "shortLink": false,
  "tags": [],
  "posts": [
    {
      "integration": {
        "id": "your-mewe-integration-id"
      },
      "value": [
        {
          "content": "Excited to share our latest update with everyone!",
          "image": []
        }
      ],
      "settings": {
        "__type": "mewe",
        "postType": "timeline"
      }
    }
  ]
}
```

### Group Post with Image

```json
{
  "type": "now",
  "date": "2024-12-14T10:00:00.000Z",
  "shortLink": false,
  "tags": [],
  "posts": [
    {
      "integration": {
        "id": "your-mewe-integration-id"
      },
      "value": [
        {
          "content": "New guide posted in the community. Check it out!",
          "image": [
            {
              "id": "guide-cover-id",
              "path": "https://uploads.postqueen.ai/guide-cover.png"
            }
          ]
        }
      ],
      "settings": {
        "__type": "mewe",
        "postType": "group",
        "group": "grp_xxxxxxxxxxxx"
      }
    }
  ]
}
```
